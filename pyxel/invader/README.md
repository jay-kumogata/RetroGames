# Invader for Pyxel/Python

## Introduction

It is a Space Invader(Taito/1978) like game.

<img src="https://github.com/jay-kumogata/RetroGames/blob/main/pyxel/invader/screenshots/invader01.gif" width="240">

## How to Play

Please execute the following from the Pyxel (version 2.5.9) environment.
Or you can play [here](https://kitao.github.io/pyxel/wasm/launcher/?run=jay-kumogata.RetroGames.pyxel.invader.invader&packages=numpy).

	> python invader.py
