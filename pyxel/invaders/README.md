# Space Invaders for Pyxel/Python

## Introduction

It is a Space Invaders(Taito/1978) like game.

## How to Play

Please execute the following from the Pyxel (version 2.5.9) environment.
Or you can play [here](https://kitao.github.io/pyxel/wasm/launcher/?run=jay-kumogata.RetroGames.pyxel.invaders.invaders&packages=numpy).

	> python invaders.py
